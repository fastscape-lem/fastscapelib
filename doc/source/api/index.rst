C++ API Reference
=================

.. note::

   We use the following conventions for template parameter names:

   - Parameters with the ``X`` prefix refer to any xtensor_ array type
     (e.g., :cpp:type:`xt::xtensor <xt::xtensor>`,
     :cpp:type:`xt::xarray <xt::xtensor>`) or expression (i.e.,
     :cpp:class:`xt::xexpression <xt::xexpression>`). This is usually
     followed by the (possibly truncated) name of the corresponding
     function argument in lowercase.

.. _xtensor: https://xtensor.readthedocs.io/en/latest/
